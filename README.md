# ja-shukujitu-to-asana

Googleの日本の祝日カレンダーのイベントを指定したAsanaプロジェクトにマイルストーンとして作成します。

## usage

```sh
npm install
export ASANA_PROJECT_IDS=123456789,987654321
export ASANA_TOKEN=your_pat
node index.mjs
```
